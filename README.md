# The Auto Wrapper
A web app that wraps your keywords so they can be used in Google Ads built using React and Material UI. A working version is hosted on Heroku at https://rad-wrapper.herokuapp.com/ 
## Installation

```
Clone or fork repo
cd into the-auto-wrapper
run npm install
```

## Usage

```
cd into the-auto-wrapper
run npm start
```
